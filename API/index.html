API Testing

API testing is a type of software testing that analyzes an application program interface (API) to verify it fulfills its expected functionality, security, performance and reliability.

Tests are performed directly on the API and as part of integration testing.

Wire Mocking

Wire mocking is a technique for testing code that communicates with external web services or APIs.

It intercepts HTTP requests and returns predefined responses, allowing you to test your application in isolation.
