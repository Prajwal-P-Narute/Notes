1Download the Installer

Go to the official Node.js website:

üëâ https://nodejs.org

Choose the LTS (Long-Term Support) version ‚Äî it's more stable for most users.

For example: Node.js 20.x.x LTS

This installer includes both Node.js and npm.

2Run the Installer

  1. Double-click the downloaded .msi file
  2. Click Next and accept the license agreement
  3. Keep the default settings (recommended)
  4. Allow it to install necessary tools (like npm, Node.js, and optionally Chocolatey)
  5. Finish the installation

3Verify Installation

Open Command Prompt or PowerShell and run:

node -v
npm -v

You should see version numbers like:

v20.12.2
10.5.0

üîç Verify Installation

Check Node.js Version

Check npm Version

Tip: If these commands don't work, you may need to restart your terminal or command prompt after installation.

üîÑ Optional: Update npm

Update to the latest npm version

Note: The -g flag installs npm globally, making it available system-wide.
