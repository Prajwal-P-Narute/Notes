Star Pattern Programs

Explore various star patterns implemented in Java

Main Program

This is the main program that calls all the different star pattern methods with a size of 5.

StarPatterns.java
public class StarPatterns {
    public static void main(String[] args) {
        int size = 5;
        
        rightTriangle(size);
        invertedRightTriangle(size);
        pyramid(size);
        diamond(size);
        hollowSquare(size);
        rightPascal(size);
        sandglass(size);
    }
    
    // All the pattern methods would be placed here
}

Right Triangle Pattern

This method prints a right-angled triangle pattern with the given size.

StarPatterns.java
public static void rightTriangle(int n) {
    System.out.println("Right-Angled Triangle (Size: " + n + "):");
    for (int i = 1; i <= n; i++) {
        for (int j = 1; j <= i; j++) {
            System.out.print("* ");
        }
        System.out.println();
    }
    System.out.println();
}

public static void invertedRightTriangle(int n) {
    System.out.println("Inverted Right-Angled Triangle (Size: " + n + "):");
    for (int i = n; i >= 1; i--) {
        for (int j = 1; j <= i; j++) {
            System.out.print("* ");
        }
        System.out.println();
    }
    System.out.println();
}

Pyramid Pattern

This method prints a pyramid pattern with the given size.

StarPatterns.java
public static void pyramid(int n) {
    System.out.println("Pyramid Pattern (Size: " + n + "):");
    for (int i = 1; i <= n; i++) {
        // Print spaces
        for (int j = i; j < n; j++) {
            System.out.print(" ");
        }
        // Print stars
        for (int k = 1; k <= (2*i-1); k++) {
            System.out.print("*");
        }
        System.out.println();
    }
    System.out.println();
}

Diamond Pattern

This method prints a diamond pattern with the given size.

StarPatterns.java
public static void diamond(int n) {
    System.out.println("Diamond Pattern (Size: " + n + "):");
    // Upper pyramid
    for (int i = 1; i <= n; i++) {
        for (int j = i; j < n; j++) {
            System.out.print(" ");
        }
        for (int k = 1; k <= (2*i-1); k++) {
            System.out.print("*");
        }
        System.out.println();
    }
    // Lower pyramid
    for (int i = n-1; i >= 1; i--) {
        for (int j = n; j > i; j--) {
            System.out.print(" ");
        }
        for (int k = 1; k <= (2*i-1); k++) {
            System.out.print("*");
        }
        System.out.println();
    }
    System.out.println();
}

Hollow Square Pattern

This method prints a hollow square pattern with the given size.

StarPatterns.java
public static void hollowSquare(int n) {
    System.out.println("Hollow Square Pattern (Size: " + n + "):");
    for (int i = 1; i <= n; i++) {
        for (int j = 1; j <= n; j++) {
            if (i == 1 || i == n || j == 1 || j == n)
                System.out.print("* ");
            else
                System.out.print("  ");
        }
        System.out.println();
    }
    System.out.println();
}

Sandglass Pattern

This method prints a sandglass pattern with the given size.

public static void sandglass(int n) {
    System.out.println("Sandglass Pattern (Size: " + n + "):");
    for (int i = n; i >= 1; i--) {
        for (int j = i; j < n; j++) {
            System.out.print(" ");
        }
        for (int k = 1; k <= (2*i-1); k++) {
            System.out.print("*");
        }
        System.out.println();
    }
    for (int i = 2; i <= n; i++) {
        for (int j = i; j < n; j++) {
            System.out.print(" ");
        }
        for (int k = 1; k <= (2*i-1); k++) {
            System.out.print("*");
        }
        System.out.println();
    }
    System.out.println();
}
