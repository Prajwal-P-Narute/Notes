Java Min/Max Finder

Multiple approaches to find minimum and maximum values in Java

Iterative Approach

O(n) Time

Traverse through the array while tracking the current min and max values. Most efficient approach with linear time complexity.

MinMaxIterative.java 
public class MinMaxIterative {
    public static void main(String[] args) {
        int[] arr = {5, 2, 9, 1, 5, 6};
        int min = arr[0];
        int max = arr[0];
        
        for (int num : arr) {
            if (num < min) min = num;
            if (num > max) max = num;
        }
        
        System.out.println("Min: " + min + ", Max: " + max);
    }
}

Using Arrays.sort()

O(n log n) Time

Sort the array and pick the first (min) and last (max) elements. Simple but less efficient due to sorting overhead.

MinMaxSort.java 
import java.util.Arrays;

public class MinMaxSort {
    public static void main(String[] args) {
        int[] arr = {5, 2, 9, 1, 5, 6};
        Arrays.sort(arr);
        int min = arr[0];
        int max = arr[arr.length - 1];
        System.out.println("Min: " + min + ", Max: " + max);
    }
}

Using Java Streams

O(n) Time

Use Stream API's min() and max() methods (Java 8+). Clean and functional approach with linear time complexity.

MinMaxStream.java 
import java.util.Arrays;

public class MinMaxStream {
    public static void main(String[] args) {
        int[] arr = {5, 2, 9, 1, 5, 6};
        int min = Arrays.stream(arr).min().getAsInt();
        int max = Arrays.stream(arr).max().getAsInt();
        System.out.println("Min: " + min + ", Max: " + max);
    }
}

Using Collections

O(n) Time

Convert the array to a list and use Collections.min() and Collections.max(). Convenient for List objects.

MinMaxCollections.java 
import java.util.Arrays;
import java.util.Collections;
import java.util.List;

public class MinMaxCollections {
    public static void main(String[] args) {
        Integer[] arr = {5, 2, 9, 1, 5, 6};
        List<Integer> list = Arrays.asList(arr);
        int min = Collections.min(list);
        int max = Collections.max(list);
        System.out.println("Min: " + min + ", Max: " + max);
    }
}

Divide and Conquer

O(n) Time

Recursively split the array and compare min/max of halves. Educational approach with linear time complexity.

MinMaxRecursive.java 
public class MinMaxRecursive {
    static class Pair {
        int min;
        int max;
    }
    
    static Pair getMinMax(int[] arr, int low, int high) {
        Pair result = new Pair();
        if (low == high) {
            result.min = arr[low];
            result.max = arr[low];
            return result;
        }
        
        int mid = (low + high) / 2;
        Pair left = getMinMax(arr, low, mid);
        Pair right = getMinMax(arr, mid + 1, high);
        
        result.min = Math.min(left.min, right.min);
        result.max = Math.max(left.max, right.max);
        return result;
    }
    
    public static void main(String[] args) {
        int[] arr = {5, 2, 9, 1, 5, 6};
        Pair result = getMinMax(arr, 0, arr.length - 1);
        System.out.println("Min: " + result.min + ", Max: " + result.max);
    }
}

Using IntSummaryStatistics

O(n) Time

Use IntStream summaryStatistics() to get min, max, count, sum and average all at once.

MinMaxStatistics.java 
import java.util.Arrays;
import java.util.IntSummaryStatistics;

public class MinMaxStatistics {
    public static void main(String[] args) {
        int[] arr = {5, 2, 9, 1, 5, 6};
        
        IntSummaryStatistics stats = Arrays.stream(arr).summaryStatistics();
        int min = stats.getMin();
        int max = stats.getMax();
        
        System.out.println("Min: " + min + ", Max: " + max);
        System.out.println("Sum: " + stats.getSum());
        System.out.println("Average: " + stats.getAverage());
    }
}
