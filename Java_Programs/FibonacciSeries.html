Fibonacci Series Program

Generate Fibonacci series using both for loop and while loop in Java

Java Program

The Fibonacci sequence is a series of numbers where each number is the sum of the two preceding ones, usually starting with 0 and 1. In this program, we start with 1 and 2, then generate the next 4 numbers in the sequence using both for loop and while loop approaches.

FibonacciSeries.java
package numbers;

public class FibonacciSeries {

    public static void main(String[] args) {
        System.out.println("========================");
        FibonacciSeries.byForLoop();
        System.out.println("========================");
        FibonacciSeries.byWhileLoop();
        System.out.println("========================");
    }
    
    public static void byForLoop() {
        int n1 = 1, n2 = 2, n3;
        
        System.out.println(n1);
        System.out.println(n2);
        
        for (int i = 1; i <= 4; i++) {
            n3 = n1 + n2;
            System.out.println(n3);
            
            n1 = n2;
            n2 = n3;
        }    
    }
    
    public static void byWhileLoop() {
        int n1 = 1, n2 = 2, n3, n = 4;
        
        System.out.println(n1);
        System.out.println(n2);
        
        while(n > 0) {
            n3 = n1 + n2;
            System.out.println(n3);
            
            n1 = n2;
            n2 = n3;
            n--;
        }
    }
}

Program Output

======================== 1 2 3 5 8 13 ======================== 1 2 3 5 8 13 ========================

How the Program Works:

  • The program starts by printing a separator line "========================"
  • The byForLoop() method is called which generates the Fibonacci series using a for loop
  • It initializes n1=1 and n2=2 as the first two numbers
  • The for loop runs 4 times, each time calculating the next number as the sum of the previous two
  • After the for loop, another separator line is printed
  • The byWhileLoop() method is called which generates the same series using a while loop
  • It uses the same logic but implemented with a while loop that runs while n > 0
  • Finally, the last separator line is printed
