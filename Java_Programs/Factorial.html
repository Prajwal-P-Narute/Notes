Factorial Number Program

Calculate the factorial of a number using Java

Java Program

The factorial of a non-negative integer n is the product of all positive integers less than or equal to n. It is denoted by n! and is used in permutations, combinations, and other mathematical calculations.

FactorialProgram.java
public class FactorialProgram {
    public static void main(String[] args) {
        int factorialNum = 3;
        int sum = 1;
        
        for (int a = factorialNum; a >= 1; a--) {
            sum = sum * a;  
        }
        
        System.out.println("Factorial of " + factorialNum + " is: " + sum);
    }
}

Program Output

Factorial of 3 is: 6
