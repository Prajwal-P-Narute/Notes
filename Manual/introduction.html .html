What is Software?

Software is the collection of specialized programs which take user input & generate desired output.

Software Testing Definition

Software testing is the process of verifying that a given software or application is generating the desired output or not. It is the process of checking completeness and correctness of the software with respect to customer expectations.

Resources involved in Software Development

Customer

Provides requirements and expectations

Business Analyst

Gathers and documents requirements

Developer

Develops application as per requirements

Tester

Checks correctness & completeness of functionality

End User

Ultimately uses the final product

Key Points

• Software testing ensures the application meets customer requirements

• Testing verifies both completeness and correctness

• Different stakeholders have different roles in the software development process
