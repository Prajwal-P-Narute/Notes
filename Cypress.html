üìä 4. Add Reporting Dependencies

npm install --save-dev mochawesome-merge mochawesome-report-generator open-cli rimraf concurrently
What these packages do:
  • mochawesome-merge: Merges multiple mochawesome reports into one
  • mochawesome-report-generator: Generates a beautiful HTML report
  • open-cli: Automatically opens the report in your default browser
  • rimraf: Clean up previous reports
  • concurrently: Run multiple scripts in parallel

üìú 5. Add Scripts to package.json

Add these scripts to your package.json file:

// package.json
"scripts": {
  "clean:report": "rimraf cypress/reports/html && rimraf cypress/screenshots && rimraf cypress/videos",
  "parallel:run": "cypress run --record false --reporter mochawesome --reporter-options reportDir=cypress/reports/json,overwrite=false,html=false,json=true",
  "merge:report": "mochawesome-merge cypress/reports/json/*.json > cypress/reports/json/mochawesome.json",
  "generate:report": "marge cypress/reports/json/mochawesome.json --reportDir cypress/reports/html",
  "open:report": "open-cli cypress/reports/html/mochawesome.html",
  "test": "npm run clean:report && npm run parallel:run && npm run merge:report && npm run generate:report && npm run open:report"
}
